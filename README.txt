 -------------------------------------------------------------------------------INTRODUCTION--------------------------------------------------------------------------------------

This pharmacy management system is a simple CLI based online pharmaceutical store to buy medicines, completely written in Python and MySQL

It is a unique combination of a pharmaceutical database management system and medical purchases of any user. 

Beginning with choices of various modes of view, from being

1) The admin
2) The customer

This software allows the individual to either update the stock present in the pharmacy or purchase medicines and other related utilities with ease.
This software system is extremely user-friendly and is in terms with the latest requirements of a customer. 
The appropriate pop-up messages for different scenarios help the user understand what the person has done to the database. 
This pharmacy management system thus brings in the administrative and consumer sections together.
